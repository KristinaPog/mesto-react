# Проект: Место
Учебный проект в рамках курса "Веб-разработчик" от Яндекс.Практикум на React. В текущей версии с сервера подгружаются карточки и информация о пользователе, в дальнейшем будет реализованна возможность добавлять новые карточки, менять аватар и данные о пользователе, ставить лайки и удалять свои карточки.

Ссылка на стриничку gitPages: https://kristinapog.github.io/mesto/
