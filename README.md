# Проект: Место
Учебный проект в рамках курса "Веб-разработчик" от Яндекс.Практикум на React. 
Он представляет собой веб-приложение, в котором пользователь может загружать фотографии интересных мест, добавлять им названия, а также ставить лайки. Приложение имеет адаптивный дизайн и может корректно отображаться на различных устройствах, включая мобильные устройства. Так же реализованна возможность менять аватар и данные о пользователе. 
Приложение работает в режиме реального времени, данные хранятся на сервере, доступ осуществляется с помощью API.
В дальнейшем будет реализованна возможность авторизации пользователя.

Спецификации HTML, CSS (Grid, Flex).
Язык Javascript, React.
Страница разработана по методологии компонентного подхода БЭМ (Nested).
Live валидация форм
Добавление и редактирование элементов
